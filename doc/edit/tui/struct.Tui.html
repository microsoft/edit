Struct Tui 

Source 
pub struct Tui { /* private fields */ }
Expand description

There’s two types of lifetimes the TUI code needs to manage:

  • Across frames
  • Per frame

Tui manages the first one. It’s also the entrypoint for everything else you may want to do.
